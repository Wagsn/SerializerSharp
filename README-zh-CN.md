# SerializerSharp

C# 序列化器，封装一些第三方库，提供一个通用化的方案来进行序列化操作。

# 项目结构

```
.
 - Document
 - Src
 - Tests
 - Demos
```

# 注意

VS2019将Standard项目修改为控制台应用程序，运行的时候需要手动加载第三方类库到bin目录
